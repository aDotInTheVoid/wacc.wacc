  • HTML (16K bytes) - entirely on one web page.
  • HTML - with one web page per node.
  • HTML compressed (8K gzipped characters) - entirely on one web page.
  • HTML compressed (8K gzipped tar file) - with one web page per node.
  • Info document (4K bytes gzipped tar file).
  • ASCII text (8K bytes).
  • ASCII text compressed (4K bytes gzipped).
  • TeX dvi file (8K bytes gzipped).
  • PDF file (116K bytes).
  • Texinfo source (4K bytes gzipped tar file).
